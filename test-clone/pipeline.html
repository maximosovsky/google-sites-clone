🔍 Extraction Strategy

From SingleFile HTML:

TAKE All <style> blocks — CSS fidelity
TAKE <main> / [role="main"] content area
REPLACE <img src="data:base64..."> → local /images/ path
REPLACE <iframe youtube> → clickable thumbnail
SKIP Google Sites header / navigation / footer (dead SPA)

From Puppeteer HTML:

TAKE YouTube iframe src URLs for thumbnails
TAKE Content verification (compare text length SF vs PP)
TAKE Fallback content if SF extraction fails

Output Template:

<!DOCTYPE html>
<html>
<head>
  <title>{title from page-map}</title>
  <style>{CSS from SingleFile}</style>
  <style>{our navigation CSS}</style>
</head>
<body>
  <nav class="gsclone-sidebar">{from page-map}</nav>
  <main class="gsclone-content">
    {content extracted from SingleFile main}
    {YouTube thumbnails from Puppeteer data}
  </main>
  <footer>Built with gsclone</footer>
</body>
</html>
